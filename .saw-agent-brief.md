## Interface Contracts

### EventPublisher

Core abstraction for event transport. SSE implementation wraps existing
sseBroker and globalBroker. Future Wails implementation wraps runtime.EventsEmit().


```
type Event struct {
    Channel string
    Name    string
    Data    interface{}
}
type EventPublisher interface {
    Publish(channel string, event Event)
    Subscribe(channel string) (<-chan Event, func())
}

```

### Deps

Shared dependency struct injected into service functions, replacing direct
Server field access.


```
type Deps struct {
    RepoPath   string
    IMPLDir    string
    Publisher  EventPublisher
    ConfigPath func(repoPath string) string
}

```

### SSEPublisher

SSE implementation of EventPublisher. Wraps existing sseBroker and globalBroker
to maintain current behavior while satisfying the EventPublisher interface.


```
type SSEPublisher struct { ... }
func NewSSEPublisher(broker *sseBroker, globalBroker *globalBroker) *SSEPublisher
func (p *SSEPublisher) Publish(channel string, event Event)
func (p *SSEPublisher) Subscribe(channel string) (<-chan Event, func())

```

### EventTransport

Frontend event transport abstraction. Wraps EventSource for SSE transport;
future Wails transport would wrap wails runtime events.


```
interface EventTransport {
  subscribe(channel: string, handler: (event: { name: string; data: any }) => void): () => void
  close(): void
}
function createSSETransport(): EventTransport

```

the wails build tag provides a no-op (assets served by Wails runtime).
