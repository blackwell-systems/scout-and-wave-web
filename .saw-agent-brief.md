## What to Implement
Add 5 scout automation commands to the web CLI that delegate to scout-and-wave-go packages:
1. analyze-deps (H3) — delegates to pkg/analyzer
2. analyze-suitability (H1a) — delegates to pkg/suitability
3. detect-cascades (M2) — delegates to pkg/analyzer
4. detect-scaffolds (H4) — delegates to pkg/scaffold

(validate.go, extract.go, etc.).

## Interface Contracts

### analyzer.AnalyzeDeps

Analyzes Go repository dependencies and produces dependency graph with wave candidates.
Used by analyze-deps command (H3).


```
func AnalyzeDeps(repoRoot string, targetFiles []string) (*DepGraphResult, error)

```

### analyzer.DetectCascades

Detects cascade candidates from type renames via AST analysis.
Used by detect-cascades command (M2).


```
func DetectCascades(repoRoot string, renames []RenameInfo) (*CascadeResult, error)

```

### suitability.AnalyzeSuitability

Scans codebase for pre-implementation status of requirements.
Used by analyze-suitability command (H1a).


```
func AnalyzeSuitability(requirementsFile string, repoRoot string) (*SuitabilityResult, error)

```

### commands.ExtractCommands

Extracts build/test/lint/format commands from CI configs and manifests.
Used by extract-commands command (H2).


```
func ExtractCommands(repoRoot string) (*CommandsResult, error)

```

### scaffold.DetectScaffolds

Detects shared types that need scaffold files from interface contracts.
Used by detect-scaffolds command (H4).


```
func DetectScaffolds(implPath string) (*ScaffoldResult, error)

```
